
## Laravel API developement
# Quick Install
<br />
clone the repo <br />
git clone https://github.com/tauseedzaman/laravel-api<br />
cd laravel-api<br />
composer install<br />
copy .env.example .env<br />
php artisan key:generate<br />
php artisan migrate<br />
